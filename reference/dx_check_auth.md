# Check DNAnexus Authentication

Verifies that the DNAnexus environment is properly authenticated. This
function checks if a valid authentication token is available.

## Usage

``` r
dx_check_auth()
```

## Value

Invisible TRUE if authenticated, aborts if not

## Examples

``` r
if (FALSE) { # \dontrun{
# Check authentication status
# dx_check_auth()
} # }
```
