# Check if DNAnexus is Initialized

Verifies that the DNAnexus environment has been properly initialized.
This function should be called before performing any DNAnexus
operations.

## Usage

``` r
dx_is_initialized()
```

## Value

Invisible TRUE if initialized, aborts if not

## Examples

``` r
if (FALSE) { # \dontrun{
# Check initialization status
# dx_is_initialized()
} # }
```
