# Check DNAnexus API Connectivity

Verifies that the DNAnexus API server is reachable and responding
correctly. This function performs a basic HTTP connection test to the
DNAnexus API endpoint.

## Usage

``` r
dx_check_connection()
```

## Value

Invisible TRUE if connectivity is successful, aborts on failure

## Examples

``` r
if (FALSE) { # \dontrun{
# Check API connectivity
# dx_check_connection()
} # }
```
