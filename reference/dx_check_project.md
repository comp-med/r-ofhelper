# Check DNAnexus Project

Verifies that the currently selected DNAnexus project is valid and
accessible. This function compares the cached project ID with the
current environment project.

## Usage

``` r
dx_check_project()
```

## Value

Invisible TRUE if project is valid, aborts on failure

## Examples

``` r
if (FALSE) { # \dontrun{
# Check current project
# dx_check_project()
} # }
```
