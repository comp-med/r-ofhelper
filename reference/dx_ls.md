# List Contents of DNAnexus Directory

Lists the contents of a DNAnexus directory.

## Usage

``` r
dx_ls(directory = NULL)
```

## Arguments

- directory:

  Character string specifying the directory path to list. If NULL
  (default), lists current directory.

## Value

Character vector with directory contents

``` r
if (FALSE) { # \dontrun{
# List current directory
```
