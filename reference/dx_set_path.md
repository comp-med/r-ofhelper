# Set DNAnexus Path

Sets the working directory within the DNAnexus project.

## Usage

``` r
dx_set_path(dx_path = NULL)
```

## Arguments

- dx_path:

  Character string specifying the DNAnexus project path.

## Value

Invisible TRUE if successful, aborts on failure
