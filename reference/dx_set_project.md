# Set DNAnexus Project

Selects a DNAnexus project to work with.

## Usage

``` r
dx_set_project(dx_project_id = NULL)
```

## Arguments

- dx_project_id:

  Character string specifying the DNAnexus project ID.

## Value

Invisible TRUE if successful, aborts on failure
